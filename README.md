# Node.js Pagination Project

## Overview
This project demonstrates server-side pagination using Node.js, Express, MySQL, and EJS. Users can view large datasets in smaller pages and navigate through them.

## Features
- Display 5 records per page.
- Previous and Next links.
- Page numbers for direct navigation.
- Prepared statements used for safe database queries.
